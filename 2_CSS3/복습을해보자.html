CSS(Cascading Style Sheets)

        마크업 언어가 실제로 화면에 표시되는 방법을 기술하는 언어

CSS Selector(선택자)

        현재 html문서 내에서 특정한 요소를 선택하는 문자 또는 문자열
        == 스타일을 적용하고자 하는 요소를 선택하는 방법

태그 선택자

        현재 HTML 문서 내에 같은 태그를 선택하는 선택자
        [작성법] 태그명{css코드;}
strong 태그에 작성된 내용
span 태그에 작성된 내용
  • strong 태그에 작성된 내용
  • span 태그에 작성된 내용

id 선택자

        HTML 문서 내에서 id 속성값이 일치하는 요소를 선택하는 선택자
        [작성법]
        #id속성값{css코드;}

        HTML 문서내에서 같은 calss 속성값을 가지는 태그를
        모두 선택하는 선택자
        [작성법]
        .class속성값{css 코드;}
